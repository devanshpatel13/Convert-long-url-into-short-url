# Convert long url into short url

### Create virtual environment

```
$ virtualenv env
```

### Activate virtual environment

```
$ source env/bin/activate
```

### Install requirement.txt file 
```
$ pip install -r requirement.txt
```

### Migration command

```
$ python manage.py migrate
```

```
